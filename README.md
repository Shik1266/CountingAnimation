## How to use


---

```dart
 CountingAnimation(
              value: '12,345',
              textStyle: const TextStyle(
                color: Colors.blue,
                fontSize: 20,
              ),
              useChar: true,
              scrollCount: 10,
              singleScollDuration: const Duration(milliseconds: 200),
              lastDuration: const Duration(milliseconds: 1000),
            ),
```
